## Q2. 最初に探している数値(t)が出てくる位置を取得する。

条件：

* 配列はソートされている
* 配列内の要素は同じ値のものがある
* 同じ値のなかで一番最初のindexを返す
* O(ln n)で

1. 普通に2分探索する
2. そっから左端のindexと探しだしたtで2文探索する。

だとO(ln n) + O(ln n/2) + O(ln n/4) + ...= O(ln n)になる・・・のかな
