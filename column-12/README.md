## Q3. m &lt; n/2 のときに、セットを使ったアルゴリズムのメンバーテストが平均2回以下であることを示す。

mがn/2より小さいので最大でも衝突確率は1/2となる。  
てことを考えると2回より小さくはなりそうな気がする。

平均メンバーテスト数は以下の式で求まりそう。

`1 * m/n + 2 * (m/n)^2 + 3 * (m/n)^3 ...`

これを検証してみると、やはり2に収束する！
